EMG Gesture Classification

Signal processing
61.7% same-subject accuracy (SVM, RBF kernel) — vs. 7.7% random chance across 13 classes

A full pipeline that classifies hand gestures from surface EMG signals using the public Ninapro DB1 dataset — the same underlying technology behind myoelectric prosthetic control.

  • Discovered through direct inspection of raw signal values that this dataset's electrodes hardware-rectify the signal before recording, meaning the textbook EMG bandpass filter was actively degrading performance
  • Removed filtering after confirming this with evidence, not assumption
  • Fixed a label-noise issue using majority-vote labeling instead of a single midpoint sample
  • Diagnosed and corrected a per-subject feature scaling issue affecting cross-subject accuracy
  • Tested and ruled out Random Forest and cross-subject scaling, documenting honest negative results

SkinSight

Computer vision
ResNet50 embeddings + adaptive symptom engine

A skin lesion similarity-matching tool that compares an uploaded photo against a reference database spanning five common skin conditions, using images sourced from the ISIC Archive.

  • Used ResNet50 image embeddings to measure visual similarity against the reference database
  • Narrows down a likely match through a sequence of adaptive symptom questions, not a single-shot prediction
